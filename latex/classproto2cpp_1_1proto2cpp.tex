\doxysubsection{Detailed Description}
The C++ style output is printed into standard output. 

~\newline
 There are three different logging levels for the class\+: 
\begin{DoxyItemize}
\item \#log\+None\+: do not log anything 
\item \#log\+Errors\+: log errors only 
\item \#log\+All\+: log everything
\end{DoxyItemize}Logging level is determined by {\ttfamily \#log\+Level}.~\newline
 Error logs are written to file determined by {\ttfamily \#error\+Log\+File}.~\newline
 Debug logs are written to file determined by {\ttfamily \#log\+File}.
